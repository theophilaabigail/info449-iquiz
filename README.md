# iQuiz

A multiple-choice quiz application for iOS.

## Part 1 Features

- Display a list of quiz topics in a TableView
- Three initial quizzes: Mathematics, Marvel Super Heroes, and Science
- Each quiz cell displays an icon, title, and description
- Settings button in the toolbar
- In-memory data storage

## Project Structure

- `Quiz.swift` - Data model for quizzes
- `QuizViewController.swift` - Main view controller with TableView
- `QuizTableViewCell.swift` - Custom table view cell
- `AppDelegate.swift` - App entry point

## How to Run

1. Open the project in Xcode
2. Select a simulator or device
3. Build and run (Cmd + R)
